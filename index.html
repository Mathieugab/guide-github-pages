Que contiendra votre guide ?

Le guide va accueillir, informer, accompagner vos voyageurs Airbnb dans leurs langues

Message de bienvenue

Laissez un petit message de bienvenue aux voyageurs qui viennent chez vous. Pratique surtout si vous ne pouvez pas les acceuillir en personne.

Connexion internet et WIFI

Les codes WIFI. C'est souvent la première chose que les voyageurs demandent quand ils arrivent dans votre logement.

Directions

Les voyageurs vont souvent vous demander quel est le moyen le plus rapide de se rendre chez vous. Selon les lieux d'arrivée BNB GUIDE vous permet de donner les meilleurs itinéraires pour arriver au logement.

Les heures d'arrivée pour le check-in

Il est toujours bon de rappeller les horaires de check-in et le process d'entrée dans l'appartement.

Aide sur les équipements

Comment faire marcher la télé ? Ou se trouve le fer à repasser ? Répondez à toutes les questions sur les équipements de votre logement.

Les activités à proximité du logement

Conseillez à vos voyageurs les activités à faire autour de chez vous ?

Les restaurants conseillés

Donnez à vos invités vos recommandations sur les bons restaux autour de chez vous.

Ou faire ses courses

Selon l'adresse de votre logement, BNB GUIDE donnera les endroits les plus proches pour aller faire ses courses.

Ou se trouve la pharmacie la plus proche

BNB GUIDE va aussi donner à vos voyageurs des indications sur la pharmacie de garde la plus proche.

Les numéros utiles et numéros d'urgence

Selon la nationalité des voyageurs, les numéros d'urgence peuvent être différents.

Les règles de vie du logement

Votre guide est un bon moyen de rappeller les règles du logement à vos invités.
