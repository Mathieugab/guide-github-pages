Aide pour la création de votre BNBGUIDE

Nous allons ici vous aider à créer et partager le guide qui sera consulté par les voyageurs qui viennent chez vous.

Répondre aux questions

Suivez le formulaire de création du guide pour répondre aux différentes questions. Nous allons vous aider à répondre à chaque question.

Ecrivez un petit message d'accueil de quelques phrases pour vos voyageurs Airbnb. Souhaitez leur la bienvenue dans votre ville et dans votre appartement. Remerciez les chaleureusement d'avoir choisi votre location Airbnb. Expliquez leur l'intérêt de ce guide et ce qu'ils vont trouver dedans.

Dans cette section vous avez juste à inscrire le nom de votre réseau WIFI puis le mot de passe de votre wifi à la question suivante. Le wifi est important pour les voyageurs Airbnb. La première chose demandée par les voaygeurs quand ils rentrent dans un logement est la plupart du temps les codes de connexion à internet.

Pour le check-in (entrée dans le logement) : vous pouvez expliquer si cela n'a pas été déja fait lors des échanges précédents comment le voyageur va accèder à votre logement à son arrivée. Cela peut être l'occasion de redonner le code d'accès de votre immeuble. Vous pouvez préciser qui sera la personne qui donnera les clés ou comment se les procurer. Si vous avez des horaires pour le check-in, n'oubliez pas de les préciser. Pour le check-out (sortie et êtat des lieux du logement), précisez ce que doit faire le locataire Airbnb dans l'appartement. Doit t'il vider les poubelles ? S'assurer que le chauffage a bien été coupé ? Préciser si un êtat des lieux aura lieu et si il y a des horaires à respecter pour libérer le logement pour les voyageurs Airbnb suivants.

Vous avez peut être déjà surement eu des questions de voyageurs pour faire fonctionner des appareils chez vous. Ou vous savez que certains objets ne sont pas faciles à mettre en route. Ce guide est l'occasion d'anticiper toutes les questions de vos futurs voyageurs Airbnb. Préciser si il le faut : comment mettre en route votre douche ou votre lave vaisselle ou lave linge (quel mode choisir, ...). Comment allumer le chauffage dans le logement ou mettre en route votre télé.

Ce guide est un bon moyen de rappeller à vos voyageurs le réglement intérieur de votre location. Les points à évoquer ici sont les suivants : Le logement est t'il fumeur/non fumeur/possibilité de fumer seulement à la fenêtre. Les soirées sont t'elles autorisées ? A partir de quelle heure faut t'il arrêter la musique dans la location Airbnb par respect pour les voisins ? Les animaux sont t'ils autorisés ?

Cela nous permettra de proposer des activités ou des recommandations personnalisées dans votre quartier à vos voyageurs Airbnb.

Vous avez surement des recommandations à faire à vos voyageurs. Un restaurant ou un café favori aux alentours. De choses à faire le premier jour lorsque les voyageurs sont fatigués. Une bonne boulangerie ? Toutes vos recommandations seront appréciées des voyageurs.

Si votre logement est grand ou possède de nombreux rangements il y a de fortes chances que vos voyageurs ne trouvent pas vos équipements. Précisez leur ici ou se trouvent les différents objets qu'ils pourraient utiliser (séche-cheveux, fer à repasser, table à repasser). Si vous avez d'autres informations essentielles à transmettre à vos voyageurs, vous pouvez les rajouter dans cette section.

Vous pouvez laisser sur ce guide un numéro ou les coordonnées d'une personne de confiance. Cela peut être votre concierge Airbnb, votre voisin, vous même. Préciser les différents moyens de vous joindre (téléphone, Whatsapp, ...)

Partager et diffuser son guide

Une fois que vous avez répondu aux questions, vous n'avez plus qu'à préciser votre mail. Vous recevez ensuite par mail tous les éléments pour partager votre BNBGUIDE.

Une fois que vous avez répondu à toutes les questions pour créer votre guide vous recevez le lien pour générer le BNBGUIDE de votre logement. Ce lien permet à n'importe quel voyageur de recevoir votre guide par mail dans la langue de son choix. Pour le moment, nous pouvons envoyer votre BNBGUIDE en Français, Anglais, Espagnol, Allemand, Russe, Portuguais, Chinois. Vous recevez aussi le QRcode de votre BNBGUIDE. Ce QRcode lorsqu'il est scanné permet aux voyageurs de facilement télécharger votre guide dans la langue de leur choix. Vous recevez aussi une affiche avec les codes du wifi et le lien vers votre guide ainsi qu'un accroche porte de bienvenue.

Il est possible de partager votre BNBGUIDE avant l'arrivée du voyageur. Pour cela il vous suffit de suivre le lien de téléchargement du guide, choisir la langue de votre invité et renseigner son adresse email.

Le plus simple est de placer un ou plusieurs supports vers votre guide. Vous pouvez par exemple imprimer un accroche-porte en y rajoutant un petit mot de bienvenue manuscrit, ou coller votre QR code de bienvenue sur votre porte. Il est aussi possible de placer le flyer en évidence sur la table de la salle à manger ou de la chambre. Vous pouvez créer votre propre support sur Word ou Powerpoint en utilisant votre QRcode ou votre lien personnel !
